Home networking is the process of interconnecting all of the devices in a home together and with the Internet in general. Home networking is used to establish a home network among the typical computing devices found in homes or for domestic use

1. MODEM

A modem is device or program that enables data to transmit over, for example cable or telephone lines. Computer information is stored digitally, where as information transmitted over telephone line is transmitted in the form of analog waves. A modem converts between these two forms.

2. ROUTER

A router is a device that forwards packets along networks to other routers over a network. A router is connected to at least two networks, commonly two LANs or WANs. Located at gateways, the places where two or more networks connect.

3. SWITCH

A network switch is a computer-networking device that connects devices together on a network by using packet switching to receive, process and forward data to the destination device.

4. W.A.P.

A wireless access point is a device that allows wireless devices to connect to a wired network using Wi-Fi, or related standards. The acess point usually connects to a router (via a wired network) as a standalone device, but it can also be an integral component of the router itself.

5. N.A.S.

Network Attached Storage is file server that connects to the network. A NAS device contains a slimmed-down operating system and file system and processes only I/O requests by supporting the popular file sharing protocols
